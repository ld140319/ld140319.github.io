${str:start}

expr substr $str start length(length不能够省略)

字符串删除

要删除的子字符串必须是字符串的开头或结尾,否则删除无效

从开始处删除

${str#substring}(从str开头开始删除最短的substring匹配)

${str##substring}(从str开头开始删除最长的substring匹配)

从结尾处删除

${str%substring}(从str结尾开始删除最短的substring匹配)

${str%%substring}(从str结尾开始删除最长的substring匹配)

字符串替换

替换第一个目标子字符串:${str/substring/replace}

替换所有目标子字符串:${str//substring/replace}

从字符串开始的位置匹配子串的长度

expr match "$string" '$substring' $substring是一个正则表达式

expr "$string" : '$substring' $substring是一个正则表达式

默认值

${parameter-word} 若parameter变量未定义，则扩展为word。
${parameter:-word} 若parameter变量未定义或为空，则扩展为word。

取变量名称列表

规则：${!开头字符串@}或${!开头字符串*}
含义：把所有以指定字符串开头的变量名列出，各变量之间按空格分开

取传入的位置参数

命令行参数：$0表示命令本身，$1表示第一个参数，$2表示第二个参数，依此类推，$@表示所有的参数
位置参数有：$0、$1、$2、$@

规则1：${@:起点}
含义：表示由起点开始，取得后面所有的位置参数

例2：

test.sh

规则2：${@:起点:个数}
含义：表示由起点开始，取得指定个数的位置参数

例3：

test.sh
